# VueJS-Calculator

This project is a calculator made with VueJS from Cod3r's Udemy course.



## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
